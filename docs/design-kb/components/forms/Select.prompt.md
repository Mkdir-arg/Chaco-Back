**Select** — native dropdown matching Input's styling (label, brand focus ring, custom chevron).

```jsx
<Select label="Programa" placeholder="Seleccioná…"
        options={['Tarjeta Alimentar', 'Becas Progresar', 'Potenciar Trabajo']} />
```

Props: `label`, `value`/`defaultValue`, `options` (string[] or {value,label}[]), `placeholder`, `error`, `helper`, `required`, `disabled`, `onChange`.
