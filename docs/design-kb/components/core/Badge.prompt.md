**Badge** — compact pill for record states, counts and labels; soft tinted background with a colored label.

```jsx
<Badge variant="success" dot>Activo</Badge>
<Badge variant="warning">Pendiente</Badge>
<Badge variant="danger" icon="fas fa-circle-exclamation">Vencido</Badge>
<Badge variant="brand">Programa social</Badge>
```

Variants: `neutral` `brand` `info` `success` `warning` `danger` `solid`. Sizes: `xs` `sm` `md`. Optional `dot` and `icon`.
