**Alert** — inline message banner with soft tinted background, leading icon, optional title and dismiss.

```jsx
<Alert variant="success" title="Trámite enviado">Recibirás una respuesta en 48 hs.</Alert>
<Alert variant="danger">El DNI ingresado no es válido.</Alert>
```

Variants: `info` `success` `warning` `danger`. Props: `title`, `icon` (override), `onClose`.
