## :material-clipboard-check-outline: Criterio de publicación

- Se publica solo lo **confirmado**. Si un dato no fue acordado, se indica como pendiente y no se inventa.
- Lenguaje orientado al cliente: qué hace y para qué sirve, no cómo está construido.
- No se publica detalle interno (estado del código, impacto técnico, riesgos ni preguntas abiertas).
- Cada funcionalidad se agrega a esta tabla y al menú del sitio (`mkdocs.yml`).
